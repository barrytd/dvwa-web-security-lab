## Lab Reports

### [2026-02-16 – Foundational Vulnerabilities](labs/2026-02-16-foundational-vulnerabilities/README.md)

This lab focused on core OWASP Top 10 vulnerabilities at Low security level, including SQL injection (error-based and blind), reflected and stored XSS, CSRF, command injection, broken access control, IDOR, and local file inclusion (LFI). The assessment emphasized exploitation techniques, impact analysis, and identification of root causes.
